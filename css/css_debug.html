CSS 3 - Debugging :

Debugging :

Nous avons vu des outils permettant de valider aussi bien le code HTML que le CSS. Cependant, il est églement utile de disposer d'un outil de debugging afin par exemple de comprendre pourquoi un style ne s'applique pas sur une balise.

Les navigateurs Firefox et Chrome disposent tous les deux d'un module destiné au développement. Vous pourrez ainsi plus facilement voir quel style s'applique sur telle balise ou encore voir le résultat de votre site sur un écran de la taille d'un smartphone.

Pour afficher dans Firefox ou Chrome le module de développement, allez simplement dans "Développement Web" → "Outils de développement". L'onglet "Inspecteur" vous permet de voir l'HTML et le CSS associé à la page courante.

Vous pouvez également tester du code HTML et CSS sur le site W3Schools. N'hésitez pas à regarder les exemples et à les tester ! Vous pouvez ensuite les incorporer et les adapter à votre sauce dans votre site.
